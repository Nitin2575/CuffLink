# MVP Plan

## MVP 1: Digital Profile

A simple iPhone-first profile that includes:

- Name
- Photo
- Bio
- Contact information
- Social links
- Current project or role
- QR code

## MVP 2: Wallet Pass

A lightweight Wallet pass that allows users to quickly share their CuffLink identity.

## MVP 3: Context Stamp

A simple feature that lets users save where they met someone, what they talked about, and why they should follow up.

## MVP 4: Follow-Up Memory

A basic reminder system that helps users act on meaningful interactions.

## Future Feature: Interactive CuffLink Map

A private map showing where connections happened, with event pins, connection clusters, and follow-up reminders.

## Success Criteria

The MVP is successful if users understand that CuffLink is not just a digital business card.

It is a way to remember real-world connections.
